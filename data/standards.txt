This is a text file showing the different json formats used in this project

characteristics :
weigth is low == less likely (weighted random)
tags are snake_case
modifiers can be beneficial (1), neutral (0), or harmful (-1)
stats are signed integers :
strength
agility
fortitude
willpower
perception
wisdom
magic

dialogs:
prefixes: Q = multi-choice question, I = information, E = text entry, N = number entry, Test = for testing purposes

terrain visibility is percentage
